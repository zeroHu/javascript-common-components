这是dialog 插件页面

js插件项目地址

先行体验

(1)点击确认或者取消反馈:

可配置参数说明

参数： { id: "dialog-wrapper",//定义 id name msg: "",//提示message cls: "dialog-wrapper",// class name closebtn: ".close",//关闭class name auto: false,//是否自动消失 modal: true,//是否需要 mask 灰色透明蒙层 buttons: { // 点击确定回调函数 openfun: function() { console.log('你点击了确定'); }, // 点击取消回调函数 closefun: function() { console.log('你点击了取消'); } } }

调用方式说明

$.Dialogs(Object); 这里的Object 就是上面👆的参数组成的对象

调用示例

第一个调用dailog

$.Dialogs();

第二个调用dialog

$.Dialogs({ msg: "hello world i am zero", buttons: { openfun: function() { console.log('i am open done'); $('.clickfun span').html('点击了确认'); }, closefun: function() { console.log('i am close done'); $('.clickfun span').html('点击了取消'); } } });

第三个调用dialog

$.Dialogs({ msg:'我是dialog3', auto:true, mask:false });

第4个调用dialog

$.Dialogs({ msg:'我是dialog4', buttons: { openfun: function() { console.log('i am open done'); $('.clickfun span').html('点击了确认'); } } });
